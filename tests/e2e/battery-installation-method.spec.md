# Battery installation method Review Centre regression

## Scenario: battery method issue navigates to a rendered field
1. Select Battery storage in the System step.
2. Leave Battery installation method empty on the Method step.
3. Open Review Centre and confirm Battery installation method missing appears.
4. Select the issue action and confirm focus lands on the Battery installation method field.
5. Enter Wall mounted battery installation with manufacturer clearances and ventilation requirements observed.
6. Return to Review Centre and confirm the battery method issue is gone.
7. Generate the PDF and confirm the battery installation method text appears in the Method statement section.

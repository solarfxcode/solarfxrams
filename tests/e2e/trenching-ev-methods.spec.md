# Trenching and EV charger method Review Centre regression

## Scenario: trenching and EV method issues navigate to rendered fields
1. Select Trenching and EV charger in the System step.
2. Go to the Method step and confirm Trenching method and EV charger installation method are visible.
3. Clear both method fields.
4. Open Review Centre and confirm Trenching method missing and EV charger method missing appear.
5. Select each issue action and confirm focus lands on the matching textarea.
6. Enter suitable method text for each field.
7. Return to Review Centre and confirm both issues clear immediately.
8. Confirm the PDF button is available once all mandatory fields are complete and the generated method section includes both method texts.
